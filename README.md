# nuntly-sdk-java

Official Java 21 SDK for [Nuntly](https://nuntly.com). Published as `com.nuntly:nuntly-java` on Maven Central. Spring Boot starter available as `com.nuntly:nuntly-java-spring`.

This repository will be populated by the next release. See [legacy/stainless-v0.x](https://github.com/nuntly/nuntly-sdk-java/tree/legacy/stainless-v0.x) for prior content.
